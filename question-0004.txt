
El Problema: Evaluación de Modelos mediante Umbrales de Decisión
Se desea evaluar cómo varía el F1-score de un modelo de clasificación al modificar manualmente el umbral de decisión.

Tu Misión:
Escribe una función llamada evaluar_modelo_por_umbrales(X, y, umbrales) que realice lo siguiente:
1. Partición: Divida los datos en entrenamiento y prueba (80/20) con random_state=42.
2. Entrenamiento: Entrene un modelo de LogisticRegression con los datos de entrenamiento.
3. Probabilidades: Obtenga las probabilidades de la clase positiva (1) para el conjunto de prueba.
4. Evaluación: Para cada valor en la lista 'umbrales', calcule el f1_score comparando las etiquetas reales con las predicciones generadas por dicho umbral (prediccion = prob >= umbral).

Retorno:
Un array de NumPy con los F1-scores correspondientes a cada umbral.
